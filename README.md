# Portfolio N64

Portfolio 3D interativo com Three.js.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Deploy (automático)

Push para `main` → GitHub Actions faz o build e deploy no GitHub Pages.

## Configuração do GitHub Pages

1. Vá em **Settings → Pages**
2. Em **Source**, selecione **GitHub Actions**
3. Pronto — o próximo push vai publicar automaticamente
